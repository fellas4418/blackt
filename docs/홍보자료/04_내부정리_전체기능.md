## 2. 사라져 VOCA (단어)

### 커리큘럼
- 중등 1,200 / 고등 2,000
- 70일 루틴 (주 5 신규 + 토·일 주말)
- Day 잠금/클리어, 하루 최대 3일치 진도 제한

### 학습 방식
- 5사이클/일 (주말 2사이클)
- 블링크: 뜻 → 숨김 → 재표시, 약 9초×2회전
- 3·5사이클 4지선다 테스트, 80% 미달 최후의 사이클
- 사이클 간 3분 강제 쿨타임 + 푸시
- 3일 누적 망각 차단 (2일+1일 전 오답·별표 선복습)
- 당일 완료 후 자유 복습

### 기록·보상
- 오답 자동 수집, 별표 수동
- 클리어/오답·별표 모달 (Day별)
- 누적 오답 DB → 10주 완주 누적 오답 시험지
- 70일 누적 리포트
- 7일마다 마스터 단어장 PDF (오답·별표 표시)
- 카카오 공유, 칭찬 배지 8종, 앱 공유
- 15단계 학습 안내 모달

### study.html
- CHARGE UI, TTS, 일시정지, 체크포인트 복원

### 지문 연동
- 분석 저장 단어 → VOCA 커스텀 연습 (블링크, 테스트 없음)

### 운영
- 이름+010 가입, GAS 로그, 쿠키 복원, PWA
- 관리자: 로고 3연타 (점프·쿨타임 삭제·초기화)
