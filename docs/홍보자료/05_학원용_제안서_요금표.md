## 한 줄 제안

**숙제는 학생이 찍고, 지문·시험 리포트·단어 진도는 한곳에 쌓이고, 상담할 PDF가 나옵니다.**  
ChatGPT 과제가 아닌, **수능·내신 형식**에 맞춘 영어 학습 OS + (선택) 3시간 영문법 연계.

## 패키지 (기능)

### Basic — 「숙제·상담 패키지」
- AI **지문 정석 분석** (교사·학생)
- **시험 분석 리포트** (문항·배점·오답·차트·PDF)
- **사라져 VOCA** 진도·주차 단어장 PDF
- 학생 **학습 노트** 클라우드 동기화

### Pro — 「입시·내신 심화」(권장)
- Basic 전체
- **Golden Key** (주제 키워드 + 실전 주제문 3종)
- **AI 1:1 질문** (학생·지문 연동)
- **질문 기록** 열람

### Academy+ — 「지점 프리미엄」(협의)
- Pro 전체
- **강사 온보딩** 1회 (90분)
- **원장용 이용 가이드** + 월 1회 운영 점검(3개월)
- (옵션) **강좌 단체 번들** — 된다니까 3시간 영문법

## 학원·학생 사용 구조 (목표)

1. 학원이 **학원 코드** 배포 (가입 시 입력)  
2. 학생: 지문 숙제 → 분석 → 단어 노트 저장 → VOCA  
3. 강사: 리포트·PDF로 수업·상담  
4. (선택) 구조 약한 학생 → **3시간 영문법** 안내  

※ 학원 코드·원장 대시보드는 **개발 로드맵** 항목 (파일럿은 코드 + 매뉴얼로 운영)
