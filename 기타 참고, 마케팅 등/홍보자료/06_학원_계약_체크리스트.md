# 학원 라이센스 · 계약 체크리스트 (내부용)

계약 전·후 확인용. 법률 자문은 전문가 확인 권장.

---

## A. 계약 전 — 상대·규모 확인

- [ ] 사업자등록증·상호·대표자 확인
- [ ] 지점 주소·연락처·담당 강사
- [ ] **재원생 수**(동시 상한) · 반 수 · 학년대
- [ ] Basic / Pro / Academy+ 선택
- [ ] 결제: 일시(학기) / 월 구독 / 분할
- [ ] 파일럿 여부 (할인율·기간·사례 사용 동의)

---

## B. 계약서에 넣을 핵심 조항

### 1) 라이선스 범위
- [ ] 사용 기간 (시작일·종료일·자동 갱신 여부)
- [ ] **지점 1곳** / 다지점 시 지점 목록·추가 요금
- [ ] **재원생 상한 N명** (초과 시 30일 내 협의 or 추가금)
- [ ] 허용 기능: Basic / Pro 목록 **부록으로 명시**

### 2) 이용 한도 (Pro)
- [ ] 지문 분석: 인당 월 __회 / 반당 월 __회
- [ ] AI 질문: 인당 월 __회
- [ ] Golden Key: 분석 포함 여부
- [ ] 초과 시 단가 또는 구간 상향 규칙

### 3) 계정·코드
- [ ] **학원 코드** (학생 가입 연동)
- [ ] 강사 계정 수 (예: 2명 포함, 추가 __만원/명)
- [ ] 학생·학부모는 B2C 약관 적용 (개인정보·이용약관 링크)

### 4) 금액·결제
- [ ] 공급가 · VAT · 총액
- [ ] 결제일·미납 시 **서비스 중단** 일수 (예: 14일)
- [ ] 환불: 파일럿 / 정식 / 중도 해지 **각각 규정**

### 5) 지식재산·브랜드
- [ ] 「트리거」「사라져 VOCA」 로고 사용 범위 (학원 홍보물만 등)
- [ ] **역브랜딩 금지** (학원명으로 앱 표시 불가, unless Academy+)
- [ ] 분석 결과·리포트 **저작권** (학습 목적 사용, 2차 판매 금지)

### 6) 데이터·개인정보
- [ ] 학생 데이터: **위탁 처리** 여부 (학원=교육기관, 당사=처리자)
- [ ] D1·Gemini 등 **국외 이전** 고지 (필요 시)
- [ ] 계약 종료 후 데이터 보관·삭제 기간

### 7) 서비스 수준·면책
- [ ] AI 결과 **참고용** (100% 정답 보장 없음) 문구
- [ ] 점검·장애 시 보상 (연장 일수 등) 상한
- [ ] 기능 변경·업데이트 통지 (30일 전 등)

### 8) 마케팅 (파일럿 시)
- [ ] 학원명·로고 **사례·후기** 사용 동의 범위·기간
- [ ] 학생 얼굴·실명 **비식별** 원칙

### 9) 해지
- [ ] 중도 해지: 잔여 기간 **일할 환불 여부** (통상 없음 or 50% 등)
- [ ] 학원 귀책 / 당사 귀책 구분
- [ ] 해지 후 학생 프리미엄 **개별 유지** 여부 (B2C 전환)

---

## C. 계약 후 — 운영

- [ ] 학원 코드 발급·강사에 공유
- [ ] 온보딩 일정 (90분) 확정
- [ ] 학생 안내 문구·QR (가입·코드 입력) 배포
- [ ] 재원생 등록 수 **월 1회** 스프레드시트 or 대시보드 확인
- [ ] API 사용량 모니터링 (한도 80% 시 알림)
- [ ] 3개월 파일럿 종료 → 본계약 or 해지 미팅

---

## D. 개발·제품 로드맵 (계약 시 과장 금지)

**현재 가능**
- 지문·시험 분석, VOCA, 노트 sync, is_premium/tri3, AI 질문

**협의·개발 후 가능 (약속 시 일정 명시)**
- [ ] 학원 코드 자동 좌석 카운트
- [ ] 원장 대시보드 (반·진도·리포트)
- [ ] 강사 전용 뷰어 (학생 PII 최소)
- [ ] 학원 로고 화이트라벨

## E. 내부 가격 의사결정 메모

| 항목 | 참고 |
|------|------|
| Gemini API | 인당 월 사용량 × 단가 → 마진 확인 |
| 지원 시간 | 온보딩·월 점검 → 시급 환산 |
| 파일럿 | 마진 낮아도 **사례 1건** = 마케팅 자산 |
| B2C와 충돌 | 학원 코드 없이 가입한 학생 = B2C 요금 유지 |
