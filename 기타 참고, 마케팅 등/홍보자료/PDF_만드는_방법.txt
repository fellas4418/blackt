트리거 블랙 · 홍보자료 → PDF / Word 변환 방법
========================================

■ 저장된 파일 (이 폴더)
  01_학생용_소개서.md
  02_학부모용_소개서.md
  03_학원용_소개서.md
  04_내부정리_전체기능.md
  05_학원용_제안서_요금표.md   ← 학원 영업 1페이지 + 요금
  06_학원_계약_체크리스트.md   ← 계약 전후 확인 (내부용)
  홍보자료_인쇄용.html  ← 브라우저에서 PDF 저장 (권장)

----------------------------------------
방법 1) HTML → PDF (가장 쉬움, 디자인 유지)
----------------------------------------
1. 홍보자료_인쇄용.html 을 Chrome/Edge로 연다
2. 상단 탭에서 문서 선택 (학생 / 학부모 / 학원 / 전체)
3. Ctrl + P (인쇄)
4. 대상: "PDF로 저장"
5. 여백: 기본 또는 좁게, 배경 그래픽 켜기

----------------------------------------
방법 2) Word 문서(.docx)
----------------------------------------
1. Word 실행 → 파일 → 열기
2. .md 파일 선택 (01~04)
3. 다른 이름으로 저장 → Word 문서(.docx)
4. 필요 시 PDF로보내기

----------------------------------------
방법 3) Google Docs
----------------------------------------
1. Google Drive에 .md 업로드
2. Google 문서로 열기 (또는 복사·붙여넣기)
3. 파일 → 다운로드 → PDF / Word

----------------------------------------
방법 4) pandoc (설치된 PC만)
----------------------------------------
  pandoc 01_학생용_소개서.md -o 01_학생용_소개서.pdf

----------------------------------------
디자인 팁
----------------------------------------
- 학생용: 다크·네온 톤 스크린샷 추가
- 학부모용: 밝은 배경 + 표·숫자(70일, 1200/2000)
- 학원용: 시나리오 4컷 + 비교표 강조
